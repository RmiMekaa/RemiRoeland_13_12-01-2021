OpenClassrooms - Project 13

Project Name: ArgentBank
Project Title: "Use an API to handle user accounts"


Project objectives

  • Convert the provided code base into React
  • Use the provided API to implement an account system (Find out more).

Features

  • User Sign-In
  • User Sign-Out
  • When the user is logged in, he has access to the transaction page.
  • When the user is logged in, he can edit his user name.

Project Requirements

  • The state of the app must be handled with Redux

Installation

  1. Clone this repository

  2. Install dependencies: npm install

  3. Launch the App: npm start

  4. Install and launch the delivered API to access to the database and test the app.
